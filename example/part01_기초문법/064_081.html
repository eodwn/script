1. 숫자 연산자 종류

연산자 기능 예
+ 더하기 var a = 10+20
- 빼기 var a = 10-20
* 곱하기 var a = 10*20
/ 나누기 var a = 10/20
% 나머지 var a = 10%20

%연산자 이럴때 사용해요

  • 홀/짝 구분할때 자주 사용
  • 수의배수
  • 반복문

2. 복합 연산자 종류

연산자 기능
+= a=a+10 -> a+=10;
-= a=a-10 -> a-=10
*= a=a*10 -> a*=10
/= a=a/10 -> a/=10
%= a=a%10 -> a%=10
