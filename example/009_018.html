CHAPTER 01 변수 - 변수 만들기

1. 변수의 종류

전역변수
전역에서 사용하는 데이터를 담는 변수이며 어디서든 접근해서 사용 가능
지역변수
특정 영역에서만 사용, 주로 함수 내부에서 만들어지는 변수
매개변수(파라미터)
함수 외부에서 함수 내부로 데이터를 전달
멤버변수(프로퍼티)
클래스 내부에 만들어지며 주로 객체에서 사용하는 정보를 담는 변수

2. 변수란

물건을 넣고, 뺄 수 있는 상자와 같이 변수는 데이터를 쓰고, 읽는 공간이다.

3. 변수는 이럴때 사용해요

  • 로그인 상태 유무를 변수에 저장
  • 사용자가 선택한 메뉴항목도 변수에 저장
  • 사이트 메뉴 항목(배열 변수에 저장)
  • 현재 기록 중인 점수도 변수에 저장
  • 현재 남아있는 캐릭터의 에너지도 변수에 저장
  • 쇼핑몰 장바구니에 담겨 있는 상품목록도 변수에 저장
  • 게시물 목록(배열변수에 저장)

4. 변수 만들기

변수는 "=" 연산자를 기준으로 var에 의해서 "변수이름"으로 변수가 만들어지고 이후 우측에 있는 데이터가 변수에 저장(대입) 된다. 변수는 RAM이 허용하는 공간내에서는 끊임없이 생성할 수 있다.
