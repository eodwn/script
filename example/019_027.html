CHAPTER 01 변수 - 데이터 종류
데이터형 범위 예
숫자형(Number) - var age = 30;
var weight=60.25;
문자형(String) 문자열 var name = "naver";
논리형 true,false var isLogin=false;
특수형(undefined) 변수는 선언했지만 값이 아직 할당되지 않은 상태 var naver;
특수값(null) 값이 아직 없다는 의미 일반적으로 undefined와 같은 의미로 사용 var band=null;
함수(function) - function showMenu(){}
var func=showMenu;
클래스(function) - function MyClass(){}
var myClass=MyClass;
객체(Object) - function MyClass(){}
var myClass=new MyClass

1. 숫자형

실무에서 다음과 같이 사용해요

  • 나이, 몸무게와 같은 숫자 정보를 저장할때 사용
  • 현재 기록중인 게임 점수를 저장할때 사용
  • 현재 남아 있는 에너지를 저장할때 사용
  • 메뉴에서 현재 선택된 메뉴 인덱스 값을 저장할때 사용

2. 문자형

실무에서 다음과 같이 사용해요

  • 메뉴 항목과 같은 글자 정보를 저장할때 사용
  • 아이디, 패스워드를 저장할때 사용
  • 게시물 제목, 날짜, 글쓴이 등 담을때 사용
  • 이름, 닉네임, 주소 담을때 사용

3. 논리형

실무에서 다음과 같이 사용해요

  • 로그인 상태 여부 저장할때 사용
  • 체크박스 선택 여부 저장할때 사용

4. undefined

값이 없는 경우 변수가 선언되었으나 대입이나 할당된 값이 없는 경우

5. null

아무것도 참조하고 있지 않다는 의미. 주로 객체를 담을 변수를 초기화할 때 많이 사용
